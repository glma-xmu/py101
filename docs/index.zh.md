# 人工智能程序设计

本课程是使用 Python 进行程序设计的全面入门课程，重点关注**人工智能**方向的应用。本站为中英双语 —— 可使用顶部的**语言切换**按钮在 English 与中文之间切换。

!!! tip "如何使用本站"
    使用左侧导航菜单浏览各章节。页面上的大多数代码都是**可运行的** —— 按 *Run* 即可在浏览器中执行、修改并重新运行。可使用右上角的搜索框快速跳转到任意主题。
